# amxx-score-hud
Simple, lightweight AMXX plugin for Counter-Strike 1.6 that adds 3 quality of life features:
- Killing an enemy plays a confirmation sound
- Close to radar, there is a team scoreboard

- In clutch moments, a __x v y__ message appears so You are aware how many players are alive (like in R6 Siege)
